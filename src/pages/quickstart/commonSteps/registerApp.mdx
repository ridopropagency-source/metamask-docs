### Get your client ID from the Web3Auth dashboard

Add social login and wallet connection functionality to your app with the MetaMask
Embedded Wallets SDK (Web3Auth).

Visit the Web3Auth dashboard and create a new project. You will use the client ID of the project in your
integration.

<a
  className="button button--primary button--sm"
  href="https://dashboard.web3auth.io"
  target="_blank"
  rel="noreferrer noopener">
  Go to the Developer dashboard
</a>

If you need help, visit the [Web3Auth community](https://web3auth.io/community/).

:::tip

To view the quickstart for a different platform, select the dropdown top right in the menu bar and choose
your target platform. For example, Android, Flutter, Angular, iOS, and more.

:::
