### Install MetaMask Embedded Wallets SDK

Install the MetaMask Embedded Wallets SDK (Web3Auth) package in your project. Select the tab that applies:

<Tabs
  defaultValue="demo"
  values={[
    { label: 'Demo app', value: 'demo' },
    { label: 'Set up your app', value: 'diy' },
  ]}>
  <TabItem value="demo">

Install dependencies from the project root with:

```bash
npm install
```

  </TabItem>
  <TabItem value="diy">

Install the Web3Auth SDK and the package you will use for blockchain calls (this demo uses `wagmi`):

```bash npm2yarn
npm install --save @web3auth/modal wagmi @tanstack/react-query
```
